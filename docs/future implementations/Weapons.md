# Futures implementacions (Armes)

---

## Llança Verinosa

### Estadístiques base

| Atribut              | Valor    |
| -------------------- | -------- |
| Dany                 | `baix`   |
| Probabilitat crítica | `-`      |
| Dany crític          | `-`      |
| Tipus d'arma         | `FÍSICA` |

---

### Habilitat

> Té una probabilitat d'aplicar **verí** durant **`3 torns`**, amb un dany verdader total de **`7.5%`**.

---

### Efecte de verí

- Probabilitat d'aplicació: **`35%`**
- Duració: **`3 torns`**
- Cada torn infligeix **dany verdader** proporcional a les càrregues restants.

| Torns restants | Dany verdader |
| -------------- | ------------- |
| **3 torns**    | `7.5%`        |
| **2 torns**    | `5%`          |
| **1 torn**     | `2.5%`        |

---

### Possibles millores

- Reduir el **decaïment del dany**.
- Augmentar la **probabilitat d'aplicar verí**.

---

### Possibles contrarestaments

- Convertir el **dany verdader** en **dany mitigat per la constitució**.

---

## Senyor de la Ventisca

### Estadístiques base

| Atribut              | Valor     |
|----------------------|-----------|
| Dany                 | `mitjà`   |
| Probabilitat crítica | `-`       |
| Dany crític          | `-`       |
| Tipus d'arma         | `DE RANG` |

---

### Habilitat

> Té una probabilitat d'aplicar **congelament** durant **`2 torns`**.

---

### Efecte de congelament

- Probabilitat d'aplicació: **`35%`**
- Duració: **`2 torns`**
- L'atac de l'afectat es redueix en **`15%`**.

---

### Possibles millores

- Augmentar l'efecte de **congelament**.

---

### Possibles contrarestaments

- Reduir la probabilitat de **congelament**.
